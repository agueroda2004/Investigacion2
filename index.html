Accesibilidad Web

Que es?

Se refiere a la práctica de garantizar que los sitios web, herramientas y tecnologías estén diseñados y desarrollados de manera que puedan ser utilizados por todas las personas, independientemente de sus capacidades o limitaciones.

La accesibilidad web es importante por varias razones:

  • Inclusión: Permite que todas las personas tengas acceso a la información y los servicios disponibles de internet.
  • Igualdad de oportunidad: Promueve la igualdad de oportunidad en el ámbito educativo.
  • Buena experiencia del usuario: Un sitio web accesible es mas fácil de usar para todos los usuarios.

Principios

Perceptible

La información y los componentes del sitio web deben ser perceptibles por todos los sentidos, incluyendo la vista, el oído y el tacto.

Ejemplo:

Proporcionar alternativas de texto para imágenes y videos.

Operable

La interfaz de usuario y los controles del sitio web deben ser operables y navegables por todos los usuarios.

Ejemplo:

Permitir que los usuarios naveguen por el sitio web utilizando el teclado o un lector de pantalla.

Comprensible

La información y el contenido del sitio web deben ser comprensibles para todos los usuarios, incluyendo a aquellos con dificultades de lectura o comprensión.

Ejemplo:

Utilizar un lenguaje sencillo y evitar la jerga técnica.
